PostgreSQL: Импорт

  • 2025/11/18 11:01:56 UTC
  • •

Работа с импортом баз данных в #PostgreSQL.

  • Удалить старую базу данных DB_NAME (при необходимости):
1

sudo -u 'postgres' dropdb --if-exists 'DB_NAME'
  • Создать новую базу данных DB_NAME с владельцем DB_USER:
1

sudo -u 'postgres' createdb --owner='DB_USER' 'DB_NAME'
  • Восстановить данные в новую базу данных DB_NAME под пользователем DB_USER из файла DB_NAME.sql.xz:
1

u='DB_USER'; d='DB_NAME'; f="${d}.sql"; xz -d "${f}.xz" && psql --host='127.0.0.1' --port='5432' --username="${u}" --dbname="${d}" --file="${f}" --no-psqlrc --single-transaction

Поиск

Используется нечёткий поиск (приблизительное сопоставление строк). Это метод поиска строк, которые приблизительно соответствуют поисковому шаблону.

Результаты поиска сортируются по точности совпадения с поисковым шаблоном. Результаты поиска, наиболее точно соответствующие запросу, находятся сверху, приблизительные результаты - снизу.
